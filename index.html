¿Que es el Css?


CSS (Cascading Style Sheets) es un lenguaje informático utilizado para dar forma, color y otras características visuales a una página.
Su función principal es definir cómo se presentan los documentos HTML a los usuarios, incluyendo el diseño, la tipografía, los colores y los márgenes.

¿Cómo funciona el CSS?

CSS se aplica a los elementos HTML mediante reglas que especifican las propiedades de estilo.
Estas reglas pueden estar en un archivo separado (hoja de estilo externa) o directamente en el documento HTML dentro de etiquetas (style)
o en atributos style. El navegador interpreta estas reglas y aplica los estilos correspondientes a los elementos HTML.

Propiedades y Selectores

Las propiedades de CSS controlan aspectos como el color,
el tamaño de fuente, el espaciado, los bordes y la disposición de los elementos.

Los selectores indican a qué elementos se aplicarán las reglas.
Por ejemplo: h1 { color: blue; } aplicará el color azul a todos los encabezados h1

Herencia y Cascada

La “cascada” en CSS se refiere al proceso de aplicar reglas en un orden específico.
Las propiedades se heredan de elementos padres a elementos hijos, pero también pueden ser anuladas por reglas más específicas

Ventajas de usar CSS

Separación de contenido y presentación:
CSS permite mantener el contenido (HTML) y el diseño (estilos) por separado.
Reutilización: Puedes aplicar las mismas reglas a múltiples elementos en diferentes páginas. Flexibilidad: Cambiar el diseño de un sitio web es más fácil con CSS.

Ejemplos de uso

Cambiar colores y fuentes.
Crear diseños de página (columnas, cuadrículas, etc.).
Animaciones y efectos visuales.
